Departures Board Setup
You will need a Live Departure Board (1.1) Consumer Key from Rail Data Marketplace if you want to display National Rail departures. You can register for access free of charge here. Follow the full instructions here.
If you also want to display additional Last seen at information for National Rail services, you will need a Service Details (1.1) Consumer Key from Rail Data Marketplace.
If you have a National Rail Enquiries Darwin Lite Webservice token you can still use the legacy data feed. This feed is likely to be discontinued in the future so we recommend registering with the Rail Data Marketplace (above). Enter the token exactly as received (in the format nnnnnnnn-nnnn-nnnn-nnnn-nnnnnnnnnnnn). You can currently still register for a token free of charge here.
Enter your openweathermap.org api key if you want to display station weather information. You can register for a free api key here.
Optionally, enter your Transport for London api key (app_key) if you have one (the board will operate fine without one). You can register for a free api key here.
